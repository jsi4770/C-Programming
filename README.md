# C Programming 학습 아카이브

이 저장소는 C 프로그래밍 수업에서 학습한 내용과 실습 코드, 개념 정리 및 간단한 프로젝트를 기록하기 위한 공간입니다.

## 📌 목적
- C 언어의 기초 문법과 메모리 구조 이해
- 문제 해결 능력 향상
- 학습 과정 기록 및 포트폴리오 활용

## 📂 폴더 구조

- notes/  
  → C 언어 이론 및 개념 정리

- exercises/  
  → 수업 실습 및 주차별 연습 코드

- projects/  
  → 간단한 프로젝트 및 응용 프로그램

## 📚 학습 내용

- 변수와 자료형
- 조건문과 반복문
- 함수
- 배열
- 포인터
- 메모리 관리
- 파일 입출력

## 💡 학습 방식

- 수업 내용을 기반으로 직접 코드를 작성하고 정리
- 핵심 개념은 notes에 문서화
- 실습 코드는 exercises에 주차별로 정리
- 프로젝트는 별도로 관리

## 🚀 목표

C 언어의 메모리 구조와 포인터를 이해하고,  
기초적인 시스템 수준의 프로그래밍 능력을 갖추는 것을 목표로 합니다.
